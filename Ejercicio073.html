Introducción a las sentencias "Else If"
Si tienes múltiples condiciones que necesitan ser resueltas, puedes encadenar sentencias if junto con sentencias else if.

if (num > 15) {
return "Bigger than 15";
} else if (num < 5) {
return "Menor que 5";
} else {
return "Between 5 and 15";
}

Transforma la lógica para utilizar la sentencia else if.
    Código original.
    function testElseIf(val) {
        if (val > 10) {
            return "Mayor que 10";
        }
        
        if (val < 5) {
            return "Menor que 5";
        }
        
        return "Entre 5 y 10";
        }
        
        testElseIf(7);
    
        Código resuelto
    function testElseIf(val) {
        if (val > 10) {
            return "Mayor que 10";
        }
        else if (val < 5) {
            return "Menor que 5";
        } 
        else {
            return "Entre 5 y 10";
        }  
        }
        
        testElseIf(7);
