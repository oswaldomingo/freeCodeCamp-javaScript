Introducción a las sentencias "Else"

Ejercicio 72

Cuando la condición en una sentencia if es verdadera, se ejecutará el bloque de código que va a continuación.
¿Qué sucede cuando la condición es falsa? Normalmente no debería ocurrir nada. Con la sentencia else, se puede ejecutar un bloque alternativo de código.

if (num > 10) {
return "Bigger than 10";
} else {
return "10 or Less";
}

Combina la sentencia if en una sola sentencia if/else. 
        Código original.
    function testElse(val) {
        let result = "";
        // Cambia solo el código debajo de esta línea
        
        if (val > 5) {
            result = "Bigger than 5";
        }
        
        if (val <= 5) {
            result = "5 or Smaller";
        }
        
        // Cambia solo el código encima de esta línea
        return result;
        }
        
        testElse(4);
    
        Código resuelto
    function testElse(val) {
        let result = "";
        
        if (val > 5) {
            result = "Bigger than 5";
        } else {
            result = "5 or Smaller";
        }
        return result;
        }
        
        testElse(4);
Pruebas
Aprobado: Sólo debes tener una sentencia if en el editor
Aprobado: Debes usar una sentencia else
Aprobado: testElse(4) debe devolver la cadena 5 or Smaller
Aprobado: testElse(5) debe devolver la cadena 5 or Smaller
Aprobado: testElse(6) debe devolver la cadena Bigger than 5
Aprobado: testElse(10) debe devolver la cadena Bigger than 5
Aprobado: No debes cambiar el código por encima o por debajo de los comentarios especificados.
